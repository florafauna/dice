# DICE GAME

## diceConsole.R

In R start game with `source('diceConsole.R')`.

## diceShiny.R

In R start game with `source('diceShiny.R')`.

### How to play

1. dices are rolled. Then user can roll every dice one more time.
2. select operation
3. enter numbers(s) and press enter.
